Rescheduling Mechanism

  • The algorithm first checks the list of available teachers for the selected day.
  • Absent teachers are identified and removed from the list of available teachers.
  • The algorithm tries to find replacement teachers who teach the same subject as the absent teacher.
  • If no same-subject teacher is available, it looks for any other free teacher.
  • Teachers are ranked based on the number of classes they have for the day and the number of consecutive classes.
  • If a teacher has fewer classes and fewer consecutive time slots, they are prioritized.
  • Replacement teachers are assigned based on availability, subject match, and class count.
