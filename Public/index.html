Rescheduling Mechanism

  • The algorithm will first check the list of available teachers.
  • Priority will be given to teachers with fewer classes scheduled on that day.
  • If a teacher has 3 consecutive classes, the algorithm will try to skip that teacher
    and any other available teacher will be ranked higher on the Priority Schedule.
  • The next teacher will be selected based on their number of classes for the day.
  • If there is a tie, the teacher with the fewest total weekly classes will be
    picked.
